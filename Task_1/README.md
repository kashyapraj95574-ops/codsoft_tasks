# Titanic Survival Prediction

## Overview

This project predicts whether a passenger survived the Titanic disaster
using a machine learning model.

## Files

-   main.py
-   Titanic-Dataset.csv
-   Titanic_Survival_Prediction_Project_Report.pdf

## Model

-   Dataset: Titanic Dataset
-   Accuracy: 78.77%

## How to Run

``` bash
python main.py
```
