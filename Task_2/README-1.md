# Movie Rating Prediction

## CODSOFT Internship - Task 2

### Objective

Build a machine learning model to predict IMDb movie ratings using movie
attributes.

### Dataset

IMDb Movies India.csv

### Technologies

-   Python
-   Pandas
-   NumPy
-   Scikit-learn

### Model

Random Forest Regressor

### Steps

1.  Load dataset
2.  Clean and preprocess data
3.  Encode categorical columns
4.  Train/test split
5.  Train Random Forest model
6.  Evaluate using MAE, MSE and R² Score

### Output

The model predicts movie ratings and prints evaluation metrics.
